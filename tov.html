The Rewrite Checklist

Test 1: The First-Paragraph Test

  • Does it make a specific claim in first 3 sentences?
  • Would you keep reading if you saw this on X?
  • Does it challenge conventional wisdom OR show insider knowledge?

Test 2: The Specificity Test

  • Replace "many" with actual number
  • Replace "companies are" with "Company X did Y"
  • Replace "experts say" with "I saw / I tracked / I counted"

Test 3: The Sentence Length Test

  • Read it out loud
  • If you run out of breath, it's too long
  • Aim for average 12-15 words per sentence (with some 5-word punches)

Test 4: The "So What" Test

  • Every section = one actionable insight
  • Can reader apply this tomorrow?
  • Does it change how they think about the topic?

Test 5: The Pattern Test

  • Did you spot a pattern others missed?
  • Did you name the pattern memorably?
  • Can reader use this pattern to predict what's next?
